# Sales Analytics System

A Python-based sales analytics application that processes raw sales data,
cleans and validates records, enriches data using an external API,
and generates analytical reports.

---

## Setup Instructions

1. Clone the repository
2. Install dependencies:

pip install -r requirements.txt


---

## Run Instructions

1. Navigate to src folder:

cd src

2. Run the application:


---

## Output Files

- `output/enriched_sales_data.txt` → cleaned & API-enriched sales data
- `output/sales_report.txt` → summarized sales analytics report
